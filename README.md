# Shortest distance coding challenge

To run the example:

    python3 shortest_distance.py

To use the function:

    $ python3
    >>> from shortest_distance import find_shortest_distance
    >>> find_shortest_distance('foo bar Foofoo foofoo', 'foo', 'foofoo', ignore_case=True)
    1

To run the tests:

    python3 test.py
